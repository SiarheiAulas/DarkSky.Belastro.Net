DarkSky@Belastro.Net

Цель и назначение:
Специализированный ресурс для астрономов-любителей в Беларуси, который помогает находить подходящие места для наблюдений, делиться информацией о новых локациях и общаться в сообществе. В перспективе масштабирование для использования любителями астрономии по всему миру. Его основная задача — предоставить информацию о наблюдательных площадках для астрономических наблюдений, помочь пользователям найти подходящие локации с учётом различных параметров.

На главной странице представлены:

информация о проекте;
ссылка на форум ЛА Беларуси;
список наблюдательных площадок;
список экспедиций ("одиссей") по поиску наблюдательных площадок
другие тематические разделы (в перспективе).

Интерактивные элементы:
элементы управления для работы с картой (отображение площадок, масштабирование).

Фильтрация и поиск:
на сайте реализован фильтр параметров, который позволяет выбирать площадки по заданным критериям (например, уровень засветки, особенности рельефа, открытость горизонта, транспортная доступность и расстояние от пользователя).
возможность сброса и очистки карты для удобства поиска и анализа данных.

Дополнительные возможности:
ссылки на форум и группы в популярных мессенджерах для общения любителей астрономии;
ссылки на поддержку — пользователи могут обратиться за помощью по техническим или контентным вопросам.

Особенности интерфейса:
ориентация на картографический формат представления данных (площадки отображаются на карте);
простота использования — интерфейс нацелен на быстрое нахождение нужной информации без излишней сложности.

Техническая реализация:

Фронтенд:
Maps JavaScript API -  для работы с картами (для отображения площадок на карте, масштабирования, управления слоями);
JS/Geolocation API - для получения геолокации пользователя;
Full stack фреймворк Laravel для построения шаблонов интерфейса blade и работы с формами;
HTML/CSS — базовая основа для вёрстки c незначительным использованием Bootstrap и Tailwind для некоторых готовых компонентов интерфейса.

Бэкенд:
PHP (Laravel) - для работы с формами, взаимодействия с БД, авторизации пользователей, сервиса расчета расстояния, методов фильтрации площадок, работы с загружаемыми файлами;
СУБД MySQL для хранения данных о площадках
